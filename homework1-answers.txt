	cd - going into the directory
	ls - shows the files in the directory

git commands used: 
	git config - log into git using your credentials 
	git clone - clones the github repository
	git status - checks the status of your local repository
	git add - adds a file to the list of files to be updated in the next commit
	git commit - commits the changes
	git push - pushes the changes in the local repository  to the github repo
